# tiledown.com

Public website source for TileDown.

The site is generated by the `tiledown` CLI from Markdown content in `content/`.
It is published to GitHub Pages at `https://tiledown.com`.

## Build

From this repo, with a sibling checkout of `TileDown/tile-down`:

```sh
scripts/build.sh
scripts/check.sh
```

If the engine checkout lives somewhere else, set `TILEDOWN_REPO`:

```sh
TILEDOWN_REPO=/path/to/tile-down scripts/check.sh
```

The generated site is written to `.build/website` by default. The GitHub Pages
workflow checks out `TileDown/tile-down`, builds this content on Linux, runs the
browser checks, and deploys the static artifact.
